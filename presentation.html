03 — КАК

Вайбкодинг

Вы описываете, что хотите — обычным языком. ИИ пишет код.
  • Никакого программирования. Только описание задачи.
  • ИИ понимает контекст и создаёт рабочий прототип.
  • Итерации: «это не то, измени вот так» — и он меняет.
Инструмент

Claude Code от Anthropic. $20/мес.

Мне нужен доброволец с ноутбуком!

03 — УСТАНОВКА

Шаг 1: Установить Node.js

 macOS
brew install node
Если нет brew: откройте Terminal и вставьте команду с brew.sh
▦ Windows
Скачайте установщик с nodejs.org
Нажмите Next → Next → Install
Node.js — среда для запуска JavaScript. Claude Code работает на ней.
03 — УСТАНОВКА

Шаг 2: Git и GitHub

Установить Git:
 macOS
xcode-select --install
▦ Windows
Скачайте с git-scm.com
Next → Next → Install
GitHub

Зайдите на github.com и создайте аккаунт. GitHub — это где хранится код. Бесплатно.

03 — УСТАНОВКА

Шаг 3: Установить Claude Code

npm install -g @anthropic-ai/claude-code
Одна команда. Работает на Mac, Windows и Linux.

npm — менеджер пакетов, он появился вместе с Node.js на шаге 1.

03 — УСТАНОВКА

Шаг 4: Аккаунт и подписка

  1. Зайдите на claude.ai
  2. Создайте аккаунт (Google или email)
  3. Оформите подписку Pro — $20/мес

Подписка даёт доступ к Claude Code и лучшим моделям. Можно отменить в любой момент.

03 — УСТАНОВКА

Шаг 5: Запуск

Откройте терминал (Terminal на Mac, PowerShell на Windows)
claude
  1. Браузер откроется для авторизации. Нажмите «Разрешить».
  2. Готово! Вы внутри Claude Code.
Совет

Создайте папку для проекта и запустите claude оттуда:

mkdir my-project && cd my-project && claude
03 — УСТАНОВКА

Шаг 6: Суперсилы (MCP)

MCP-серверы дают Claude Code доступ к внешним инструментам.
Firecrawl — Поиск и чтение веб-страниц
claude mcp add firecrawl -- npx -y firecrawl-mcp
Context7 — Документация библиотек — всегда актуальная
claude mcp add context7 -- npx -y @upstash/context7-mcp@latest
Есть сотни MCP-серверов для всего: базы данных, деплой, Telegram, Notion...
